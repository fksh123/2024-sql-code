**11 데이터와 정보의 차이를 구별하여 설명하시오.**

데이터 중복성 감소

데이터 동시 공유 가능
DBMS를 통해 여러 사용자가 동시에 데이터 접근 가능.

DBMS를 통해 통일된 데이터 접근 방식 적용 가능.
데이터 가용성 향상
장애 복구 기능으로 데이터 접근 안정성 보장.
응용 프로그램 개발 비용 절감
DBMS 지원으로 개발 시간과 유지보수 비용 감소.
